# University Course Registration System

A console-based university management system built in C++.

## Features
- Student registration and course enrollment
- Faculty management and grade entry
- GPA calculation with grade-point logic
- File I/O for persistent data storage

## Technologies Used
- C++ with Object-Oriented Programming
- STL containers: map, vector
- File handling (fstream)

## OOP Concepts Applied
- Inheritance, Polymorphism
- Abstract classes and virtual functions
- Constructor overloading, Encapsulation

## How to Run
g++ "OOP PROJECT 1.cpp.txt" -o university_system
./university_system
